第四部分 数据导出功能

数据导出功能

笔记导出Word/PDF · 错题本导出打印 · 纯前端实现

Word 导出原理

利用 Blob 对象 将 HTML 内容包装为 Word 格式的二进制数据， 通过 URL.createObjectURL() 生成临时下载链接， 最后用 a.download 属性触发浏览器下载。

PDF 导出原理

用 window.open() 打开新窗口写入 HTML 内容， 然后调用 window.print() 调出浏览器原生打印对话框， 用户选择"另存为 PDF"即可导出，零依赖、兼容性好。
