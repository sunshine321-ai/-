03 核心代码 · Vite 跨域代理

开发环境跨域解决方案

server: { port: 5173, proxy: { '/api': 'http://localhost:8000', '/files': 'http://localhost:8000', }, },
跨域限制只存在于浏览器 ↔ 服务器之间
服务器 ↔ 服务器之间没有同源策略限制
前端以为请求的是 5173，实际响应来自 8000

首屏性能优化策略

{ path: '/', name: 'home', component: () => import('@/views/HomePage.vue') }, { path: '/study', name: 'study', component: () => import('@/views/StudyPage.vue') }, { path: '/ai-tutor', name: 'ai-tutor', component: () => import('@/views/AiTutorPage.vue') }, // ... 共 7 个路由
首屏只加载首页 JS，其余路由按需加载
显著减少首屏白屏时间
提升用户体验和 LCP 指标

API 模块化 06 / 09
06 API 封装示例

封装带来三大好处：复用、统一、解耦

import axios from 'axios' export const getLearningProgress = async () => { const response = await axios.get(`${API_BASE}/progress`) return response.data } export const saveLearningProgress = async ( chapterKey, progress, completed ) => { const response = await axios.put(`${API_BASE}/progress`, { chapterKey, progress, completed }) return response.data }
复用：同样的请求多个组件都能用
统一：错误处理、参数格式都标准化
解耦：后端 API 改了，组件代码不用动
