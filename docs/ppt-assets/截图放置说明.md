# 答辩 PPT 截图放置说明

请将最新截图裁剪为接近 16:9 的 PNG 图片，并使用以下文件名放在本目录：

1. `01-home.png`：平台首页，建议包含导航、视频、学习进度和主要入口。
2. `02-study.png`：课后学习页，建议展示交互练习或知识可视化章节。
3. `03-playground.png`：卷积操场，建议展示原图、3×3 卷积核与处理结果。
4. `04-showcase-3d.png`：3D 卷积核展示页。
5. `05-data-viz.png`：学习数据洞察页。

放入图片后直接刷新 `docs/DEFENSE_PRESENTATION.html`，占位框会自动替换为截图，无需修改 HTML。
